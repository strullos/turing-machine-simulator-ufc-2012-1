Detailed Description

Representa uma regra do diagrama. Cada regra eh referente apenas a um unico modulo e eh composta por um conjunto de simbolos de leitura e Modulos associados. Uma regra do diagrama especifica que, especifica o proximo modulo a ser executado, de acordo com o simbolo atual na cabeca de leitura.

Obs: Para evitar a criacao de mais arquivos, essa classe foi definida toda inline.
